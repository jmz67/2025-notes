## 3.1 函数式编程是什么意思

Js 支持函数式编程，因为在 Js 中，函数是一等成员。这意味着，变量可以做到的事情函数都可以做到。最新的 Js 句法对这门语言做了一些改进，使用函数式编程更加方便了，例如箭头函数，promise 和展开运算符。
